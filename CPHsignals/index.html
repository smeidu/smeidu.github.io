CPHsignals — Send Morse Code Signals through Twitter (2012)

CPHsignals was a project to connect two Copenhagen neighborhoods using the old maritime technology of signal lamps combined with the modern network layer of Twitter messages.

The aim of this urban intervention was to connect the two sides of the harbor, which are geographically close but still feel disconnected. Furthermore, it showcased how communication between remote locations was achieved in the past in the context of Copenhagen’s maritime culture.

The installation had a graphical user interface on one side to allow users to type a message. An Arduino translated the message into Morse code and transmitted the signals using old maritime signal lamps. The same interface was placed on the receiving side to decode and display the messages.

Each of the two signal lamp stations has its own Twitter account so people can follow the conversations between the two harbor sides online. User from around the world can also use Twitter to send a new Morse code message by an @reply to one of the Copenhagen signal lamps and broadcast their message into the harbor.

#CPHsignals is a fully functional prototype and was successful in provoking interactions between the two sides while exhibited in the Copenhagen harbor. This project was done together with Ana Catharina Marques and Kostantinos Frantzis for the Systems & Layers class at CIID. Faculty: Adam Greenfield & Engin Ayaz. The signal lamps were kindly sponsored by maritime-antiques.dk. Special thanks goes to Sebastian Schuster for his development support.
