OEBB train schedule interface for iPhone

I designed and developed a iPhone/Safari mobile web optimized interface for the Austrian train timetable. Back in 2008 the iPhone was brand new, mobile web something yet to be explored. This research project was a good vehicle to explore to oppurtunities and provide something useful. I presented the prototype at a Barcamp conference and it was widely featured in Austrian media.

The prototype is still available via oebb.smeidu.com

Date: 2008
Project by: Markus Schmeiduch. This project not related to ÖBB. It only acted as an showcase for how a mobile-web/iPhone optimized UI could look like. Please don’t sue me;-)
