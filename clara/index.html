Clara platform vision

Our goal was for Flying to be the ultimate flight companion for all your air travel needs. From the moment you have booked your flight to the time you land, the app is a trusted partner. Flying brings together all the important elements of air travel to create something that feels right, works well and mostly importantly, is useful and indispensable to you.

Clara design system

is based on the Material React UI library and it's excellent Figma template. We defined the Clara-Theme to be tailored for an broad, eldery user base: clear contrast, large font sizes, simple UI building blocks, warm colors that also help to seperate between the different user types (caregiving users, dependent users, consultant users).

Clara UX

The platform and apps are currently in active development. We recently finished the initial alpha versions for the client app (Web / iOS / Android app for the caregiving relatives) and consultant app (Web UI for professionell consultation work). The alpha version was meant for small scale user tests and to learn and further develop the product. We are now gearing towards a bete release for scaled up test with invited users in 2022.
